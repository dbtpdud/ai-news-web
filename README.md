# 📰 오늘의 5분 뉴스 요약 웹

AI를 활용하여 제작한 간단한 뉴스 요약 웹페이지입니다.  
사용자는 최신 뉴스 5개를 한눈에 확인하고, 필요 시 원문 기사로 이동할 수 있습니다.

---

## 🔍 프로젝트 소개

본 프로젝트는 인공지능(ChatGPT)을 활용하여 짧은 시간 안에 웹페이지를 제작하는 것을 목표로 하였습니다.  
Google News RSS 데이터를 기반으로 최신 뉴스를 가져오고, 핵심 내용을 간단하게 보여주는 기능을 구현하였습니다.

---

## 🚀 주요 기능

- 📌 최신 뉴스 5개 자동 불러오기
- 📌 뉴스 한 줄 요약 제공
- 📌 기사 원문 링크 연결
- 📌 새로고침 버튼으로 최신 뉴스 갱신
- 📌 모바일에서도 보기 좋은 반응형 디자인

---

## 🛠 사용 기술

- HTML
- CSS
- JavaScript
- Google News RSS
- rss2json API

---

## 💡 AI 활용 방식

본 프로젝트는 다음과 같은 방식으로 AI를 활용하여 제작되었습니다.

- ChatGPT를 이용한 웹 코드 생성
- UI/UX 디자인 구조 설계
- 기능 구현 아이디어 도출
- 코드 수정 및 개선

이를 통해 웹 개발 경험이 많지 않아도 빠르게 결과물을 제작할 수 있었습니다.
